===================
Extension Challenge
===================

This extension is not an essential part of the experiment (unless your Instructor has told you otherwise);
it allows you to demonstrate defences against this type of sidechannel (although usually they should be 
designed-in to the structure of the computatin rather than applied outside as a wrapper.).

Error Wrapping
______________

**Demonstrate a Python function which provides a password check by calling pwcheck(), but prevents your previous attack from succeeding.**
